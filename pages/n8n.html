Meet n8n

A free, open-source automation canvas where you build workflows visually by connecting nodes — no code needed.

The basics

What, why and how

What is n8n?

A visual workflow builder

You drag nodes onto a canvas, connect them with lines, and n8n runs the whole thing automatically. Like LEGO bricks — each node does one job, and you snap them together to build something powerful. No code, no setup.

Why use n8n?

Free, powerful, flexible

  • 100% free & open source — no task limits
  • 400+ app integrations built in
  • Runs in the cloud or on your laptop
  • More powerful than Zapier — no paywalls
  • Your data stays private — you control it
  • 500+ ready-made workflow templates
How does it work?

Five steps to your first workflow

  1. Open n8n and create a new workflow
  2. Add a Trigger node — what starts it
  3. Add Logic and Action nodes
  4. Connect them with lines on the canvas
  5. Click Activate — runs forever

Understanding APIs

The waiter analogy

n8n talks to 400+ apps through APIs. You don't need to understand APIs — but here's an analogy that makes it click.

🧑‍💻
You (the app)
Place your order — "I want the Gmail data"
→
🤵
The API (waiter)
Carries your request to Google's servers and brings back the result
→
🍽️
The server (kitchen)
Processes the request and returns your emails as data

n8n handles all of this for you — you just pick the node, connect it, and it works.

Node types

The three types of nodes

Every node in n8n fits into one of three categories. Learn these and you understand how any workflow works.

Trigger Starts the workflow
Listens for an event. The moment it fires — n8n wakes up and runs the workflow. You can only have one trigger per workflow.
Real examples
📅 Schedule"Run every Monday at 9am"
🔗 Webhook"When a form is submitted"
📧 Gmail"When a new email arrives"
📊 Sheets"When a new row is added"
ScheduleWebhookGmailSheetsRSS FeedTelegram
Logic Makes decisions
Controls the flow — routes data down different paths based on conditions, like real IF/ELSE decisions in code (but visual).
Real examples
🔀 IF"If score > 80, send congrats"
🔁 Switch"Route by quiz score 0–2, 3–4, 5"
🔍 Filter"Only pass rows where status = active"
🔧 Set"Format the date before saving"
IFSwitchFilterSetLoopMerge
Action Does the work
Performs a task — sends a message, saves a file, updates a spreadsheet. Chain as many action nodes as your workflow needs.
Real examples
📊 Sheets"Append a new row with form data"
📧 Gmail"Send a personalised email"
💬 Telegram"Send an alert message"
☁️ Drive"Upload and rename the file"
GmailSheetsTelegramDriveNotionSlack

Why n8n over Zapier or Make?

🟢 n8n advantages

✓100% free and open source — no task limits or paywalls
✓Self-hostable — your data never leaves your machine
✓Run JavaScript inside nodes when you need real code
✓500+ workflow templates to start from
✓Active community — 30,000+ members on the forum

🔑 When to use alternatives

→Zapier — if you need the simplest possible setup with no learning curve
→Make (Integromat) — if you prefer a more visual flow diagram style
→All three connect to similar apps — the core concepts you learn today transfer across all of them
